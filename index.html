6.3 故肆 — 未建尺选项（额外兜底机制）

xxxxxxxxxx
【获取途径】
  经过不期而遇 → 大概率出现【移时换物】事件

【未建尺机制】
  每次临摹岁形时 → 遇到收益不理想的情况 → 选未建尺选项
    ├─ 每选一次消耗 1烛火

    └─ 共能选择 4 次


  4次奖励：
    ├─ 第1-3次：12源石锭 + 6生命上限 + 10护盾
    └─ 第4次（最后一次）：6希望 + 2个随机藏品


【使用策略】
  尽量在烛火全部耗尽前拿完前3次资源
  如果未建尺还没拿到 / 已拿完全部资源 → 出现收益不理想时：
    └─ 找黍溯果（黍常乐），永远不找令买藏品
          └─ 如果希望用于招募干员 → 直接结束事件


【补充：申铸轮】
  战斗掉落或商店刷新 → 藏品【申铸轮】
    ├─ 可以在黍常乐中增加额外选项
    └─ 帮助获得更多资源（出率较低，这里只提一下）

  └─ 获得的源石锭 → 投入故肆兑换生命值或护盾


核心价值：为故肆叠血提供源石锭来源


【前置条件不足时】
  生命值 < 11 + 源石锭 < 20
    ├─ 已经找到故肆 → 继续开图直到找到黍常乐
    └─ 或被【祸乱/疑/杂疑】堵住去路 → 转而去故肆
          └─ 在【筹谋】等非一次性事件节点中触发代理人选项获取资源


【黍常乐不是必出事件】
  └─ 当局没有刷新 → 在上述非一次性节点中选代理人的援助
        └─ 杂疑、祸乱 → 根据需要选获取资源、藏品或招募券
              ├─ 账号练度越高 → 越推荐选招募券
              └─ 否则 → 选择藏品


【其他常乐（不推荐）】
  ├─ 年常乐「三缺一」→ 收益不稳定
  └─ 令常乐「掷地有声」→ 收益不稳定
